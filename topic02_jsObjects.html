Object Literal

I have created two objects called employee_1 and employee_2. Please
enter either the number 1 or 2 to view a few of the object properties.


Add Properties to an Exsisting Object

We will use the employee_1 object created above and add a property
to our object. No need to re-code the literal, just add a property
by declaring our objectName.property and set it equal to the data we
wish to insert into that property. We will add a hireDate property to
employee_1.


Object.create()

We will use this method to create an object from an already exsisting
object. We will take employee_2 and use it as a prototype to create employee_3.
From there we can edit the properties, but the properties that we do not change
will remain the same as the properties from employee_2.



This Method

We have created a function as part of the employee_1 object that will
display a list of all the information we have about employee_1. We use the
'this' keyword to refer to the employee_1 object and call it's properties.
